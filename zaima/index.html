AI CLI status checker

zaima

A small Bash utility that checks local AI coding CLIs for availability, login state, and installed version.

What it checks

  • Codex, Claude Code, Gemini, and Hermes Agent command availability.
  • Whether each assistant appears authenticated in the current user environment.
  • The installed CLI version for a quick terminal health snapshot.

Run it

git clone https://github.com/gigode/zaima.git
cd zaima
./zaima

Output modes

In an interactive terminal, zaima renders a compact dashboard. In scripts or redirected output, it prints one plain-text status line per assistant.
