Terminal forum browser

4d4y_cli

A BBS-inspired command-line client for browsing the 4D4Y forum, reading threads, opening links, and replying from the terminal.

What it does

  • Lists forum boards and opens Discovery after login.
  • Shows thread lists with keyboard navigation and page controls.
  • Reads posts in terminal-sized pages and exposes images or links as selectable items.
  • Stores login configuration under the user config directory.

Install and run

git clone https://github.com/gigode/4d4y_cli.git
cd 4d4y_cli
./4d4yinstall
./4d4y

Controls

  • J/K or arrow keys move selection.
  • H/L changes pages.
  • Enter opens the selected thread, image, or link.
  • B goes back and Q exits.
